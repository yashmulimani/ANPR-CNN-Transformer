# 🚗 Number Plate Recognition System (CNN + Transformer)

## 📌 Project Overview
This project implements an end-to-end Automatic Number Plate Recognition (ANPR) system using:
- Computer Vision techniques (Thresholding, Canny, Contours)
- Deep Learning (CNN + Transformer)

---

## ⚙️ Features
- Number plate detection using CV preprocessing
- Plate extraction using ground truth annotations
- Character recognition using CNN + Transformer
- Character-level and full plate accuracy evaluation

---

## 🧠 Model Architecture
- CNN (ResNet18) → Feature Extraction
- Transformer → Sequence Modeling
- CrossEntropy Loss for prediction

---

## 📊 Results
- Character Accuracy: **80%+**
- Full Plate Accuracy: **~8%**

---

## 🛠️ Tech Stack
- Python
- OpenCV
- PyTorch
- NumPy

Indian License Plate Dataset (Kaggle)
